{{TITLE}} is a living operations manual for the engineers who build, support, and maintain {{ORG_NAME}}'s software. Read it end-to-end when you're new, and dip into it as a reference after that.

This is a living document, not a one-time artifact — it's expected to drift out of date the moment things change, so it ships with a built-in way to propose corrections. Every substantive paragraph (and every embedded diagram) can be annotated: hover over it and a suggestion button appears.

How the suggestion feature works Hover any paragraph, table, list, or code block and click the button (or the Suggest button on a diagram), then describe the change you'd like. The guide generates a ready-to-paste prompt for an agent running in this repository. The agent checks the suggestion against the source of truth, records it in the change log, edits the relevant section source, and rebuilds. Nothing is sent anywhere automatically — you stay in control of the prompt.
How this page is builtThis guide is assembled from source: each section lives in its own fragment under sections/, the structure lives in palimpsest.config.json, and the look comes from theme/tokens.css. Running pal build reassembles them into the published file — never hand-edit the generated HTML.
